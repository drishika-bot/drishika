# Drishika Parida - Portfolio Website

A modern, premium personal portfolio website for Drishika Parida, a Computer Science Engineering student.

## Features

- **Identity Card Hero**: A unique 3D-effect student badge component.
- **Editorial Design**: Luxury typography and clean layouts.
- **AI Chatbot**: A floating assistant (DrishikaBot) to help visitors learn about her skills and projects.
- **Code Viewer**: Interactive modal to view Java and Web source code directly.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.

## Project Structure

- `index.html`: Main landing page structure.
- `style.css`: Custom premium styling with animations.
- `script.js`: Interactive logic (chatbot, modal, scrolling).
- `images/`: Local folder containing profile and about images.
- `StudentManagementSystem.java`: Java project source code.
- `MatrixCalculator.java`: Java project source code.

## How to Run

1. Clone or download the repository.
2. Open `index.html` in your favorite web browser.
3. For the Java projects, compile and run them using any Java environment (e.g., `javac StudentManagementSystem.java` then `java StudentManagementSystem`).
